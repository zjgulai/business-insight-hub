# Business Insight Hub

母婴赛道 AI 消费产品洞察平台。基于深度市场研究，呈现反直觉创新方案。

## 技术栈

- React 19 + TypeScript
- Vite 7
- Tailwind CSS v3 + shadcn/ui
- Framer Motion
- React Router v7

## 目录结构

```
business_insight_hub/
├── app/          # React 前端应用
├── docs/         # 研究报告与分析文档
│   └── research/ # 维度拆解研究
└── assets/
    └── diagrams/ # 图表与可视化图片
```

## 本地开发

```bash
cd app
npm install
npm run dev
```

## 构建

```bash
cd app
npm run build
```
